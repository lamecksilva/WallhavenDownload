### WallhavenDownload

App/Script to download RANDOM wallpapers from wallhaven.cc

#### Instalation

```sh
$ git clone https://github.com/lamecksilva/WallhavenDownload.git
$ cd WallhavenDownload/
$ npm i
```

#### Example

```sh
$ node index
https://wallhaven.cc/w/838gyk
Old File Deleted
Starting Download
Download done
```
